Para executar as funções de busca em largura, abra o console do navegador (F12 no chrome).

Para criar um novo grafo gerado aleatóriamente use o comando:

> var G = new graph(< vertices>, < digraph>)

sendo G, a variavel em que o grafo será armazenado, e os parametros, vertices, um inteiro que representa o numero de vertices que terá o grafo gerado e digraph um booleano que caso verdadeiro, o grafo gerado sera orientado, e caso falso, não.

Caso deseje usar um grafo custumizado, é possivel seta-lo manualmente usando:
G.vetices = < quantidade de vertices do grafo custumizado>
G.adj = < matriz de adjacencias equivalente ao grafo desejado>

Para executar a busca em largura, deve se usar o comando:

G.BFS(< indice do vértice inicial>)
